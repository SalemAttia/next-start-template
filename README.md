# Next start template

## is a starter nextjs template including all of these project configuration

- Base project structure
- typescript configured
- eslint
- test
- Google analytic
- sentry for error mentor
- CSS module example
- global CSS
- custom error page
- custom 404 page
- circle ci configured
- sitemap script generator

## How to use

```bash
## install
npm i
```

```bash
## Development
npm run dev
```

```bash
## Bauild
npm run build
```

```bash
## Start Prod
npm start
```

```bash
## test
npm run test
```

```bash
## lint
npm run lint
```

```bash
## format your files
npm run format
```
